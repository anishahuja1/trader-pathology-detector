# Submission Guide - NevUp Behavioral Intelligence Platform

Congratulations on completing the assignment! Use this guide to ensure your submission is professional and complete.

## 📦 What to include in your Zip file
1.  **`src/`**: All source code.
2.  **`prisma/`**: Schema and the `dev.db` database file (this ensures the recruiter sees your seeded data).
3.  **`DECISIONS.md`**: Your architectural justifications.
4.  **`README.md`**: Setup and usage guide.
5.  **`nevup_postman_collection.json`**: The testing collection.
6.  **`package.json`** & **`tsconfig.json`**: Project configuration.
7.  **`generate-token.ts`**: The utility used to create test tokens.

> [!IMPORTANT]
> Do NOT include `node_modules/` or `dist/` in your zip file. They are too large and should be generated by the recruiter using `npm install`.
